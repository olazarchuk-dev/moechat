##### Welcome to Futa

### tech stack
+ **Back End**
  - Golang 1.13+
  - Fiber
  - Postgre SQL
  - Websocket
+ **Other**
  - docker
  - google cloud for the server
  - github action for ci/cd for the back end
  - vercel for deploy the front end

```shell script
> go build main.go
```
